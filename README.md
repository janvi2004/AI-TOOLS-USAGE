# AI-TOOLS-USAGE
## 📌 Project Overview
This project analyzes the usage trends of AI tools using Tableau to understand adoption and growth patterns.

## 🎯 Objectives
- Analyze AI tool usage trends
- Compare adoption across categories
- Present insights using visual dashboards

## 🛠️ Tools & Technologies
- Tableau

## 🔍 Key Tasks Performed
- Created dashboards and visualizations
- Compared usage trends across tools
- Highlighted key insights

## 📊 Results & Insights
- Identified growth trends in AI adoption
- Visualized comparative analysis of tools
- Delivered clear and interactive insights
